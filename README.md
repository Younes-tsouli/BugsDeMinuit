# Émilie

## Avertissement sur le contenu

Ce jeu traite de sujets matures incluant le cyberharcèlement, la précarité financière, la vente de contenus intimes et la détresse psychologique. Il est déconseillé aux personnes sensibles et aux joueurs mineurs.

## But du jeu

Vous incarnez un détective engagé par un responsable anonyme. On vous a fourni l'ordinateur personnel d'Émilie.

Votre objectif est de fouiller le système d'exploitation pour comprendre la vie de l'ancienne propriétaire et connaitre les raisons de son suicide.

### Aide / Spoilers

Si vous êtes bloqué dans l'enquête, voici quelques pistes pour avancer. Ne lisez ce qui suit que si nécessaire.
- Mot de passe de la session : "barnabé".
- Cliquer sur la notification Whatsapp.
- Ouvrir le journal pour trouver l'indice à la dernière page du journal.
- Ouvrir internet explorer.
- Taper "Drive" dans la barre de recherche.
- Ouvrir le fichier caché.
- Le jour le plus heureux de sa vie : "15 Juin 2024".
- Aller sur le lien OnlyPhon et voir le qu'elle a un compte Instagrom.
- Taper Instagrom dans la barre de recherche.
- Le dernier message envoyé est le stalkeur.
- Dans les commentaires de certains posts anonyme a posté un commentaire cliquer sur son profil.
- À bientôt pour la suite.
